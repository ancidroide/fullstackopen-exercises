# Full Stack Open - Part 3: Backend Development with Node.js and Express

This section focuses on developing backend services using Node.js and Express. We'll learn how to create RESTful APIs, handle HTTP requests, implement CRUD operations, and deploy applications to the internet.

## Phonebook Backend

Backend for the phonebook application developed during this part of the course.

### Live Application
[Link to the deployed backend](https://fullstackopen-exercises-dk39.onrender.com/api/persons)

### API Endpoints
- `GET /api/persons` - Retrieve all persons
- `GET /api/persons/:id` - Retrieve a specific person
- `POST /api/persons` - Create a new person
- `DELETE /api/persons/:id` - Delete a person
- `GET /info` - Application information

### Features Implemented
- Full CRUD operations for phonebook entries
- Data validation and error handling
- Proper HTTP status codes
- Integration with frontend application
- Deployment to cloud platform
