# Full Stack Open Exercises

Repository containing solutions to exercises from the [Full Stack Open course](https://fullstackopen.com/en/), a comprehensive web development course covering modern JavaScript-based web development.

## Course Overview

Full Stack Open is a free course developed by the University of Helsinki that teaches modern web development practices, including:
- React and Redux for frontend development
- Node.js and Express for backend development
- MongoDB for database management
- Testing, CI/CD, and deployment strategies
- GraphQL and other advanced topics

## Repository Structure

This repository is organized by course parts:
- `part0/` - Fundamentals of Web Apps
- `part1/` - Introduction to React
- `part2/` - Communicating with Server
- `part3/` - Programming Practices in JavaScript/TypeScript
- `part4/` - Testing Express Servers, User Administration
- `part5/` - Testing React Apps
- `part6/` - State Management with Redux
- `part7/` - React Router, Custom Hooks, Styling UI
- `part8/` - GraphQL
- `part9/` - Typescript
- `part10/` - React Native
- `part11/` - CI/CD
- `part12/` - Containers
- `part13/` - Using relational databases
