# Part 0 Exercises - Fundamentals of Web Apps

This folder contains solutions to the exercises from [Part 0: Fundamentals of Web Apps](https://fullstackopen.com/en/part0/fundamentals_of_web_apps) of the Full Stack Open course.

## Exercises Completed

- **0.4: New Note Diagram** - Sequence diagram showing the flow when creating a new note
- **0.5: Single Page App Diagram** - Sequence diagram showing how the SPA loads and fetches data
- **0.6: Creating a New Note in SPA** - Sequence diagram showing how a new note is created in the SPA version

All diagrams are created using Mermaid syntax and illustrate the communication between browser and server in different scenarios.
