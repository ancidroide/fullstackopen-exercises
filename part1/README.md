# Part 1 Exercises - Introduction to React

This folder contains solutions to the exercises from [Part 1: Introduction to React](https://fullstackopen.com/en/part1) of the Full Stack Open course.

## Overview

Part 1 introduces the fundamentals of React, including:
- Components and JSX
- Props and state
- Event handling
- Conditional rendering
- Lists and keys
- Forms and controlled components
- Debugging React applications

## Exercises

This section will contain exercises covering:
- Basic React component creation
- State management with useState hook
- Handling user input and events
- Component composition and props drilling
- Conditional rendering techniques
- Working with lists and keys
- Form handling and validation

More details and completed exercises will be added as they are completed.
